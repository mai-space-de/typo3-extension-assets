## Asset Inclusion

* Inline asset inclusion — embed CSS/JS/SCSS inline from Fluid templates
* Path-based asset inclusion — link to external or local asset files
* Deferred loading — defer or async script loading

## Compilation & Minification

* CSS/JS minification — via `matthiasmullie/minify`
* SCSS compilation — server-side via `scssphp/scssphp`; exposed as a service to other extensions
* SVG sprite building — combine multiple SVGs into a single sprite sheet

## Caching & File Management

* Caching — compiled and minified assets cached by TYPO3 caching framework
* File management — TYPO3 file list and metadata support for managed assets
